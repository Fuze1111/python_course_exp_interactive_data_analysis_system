## 任务要求
| 模块       | 必做要求                                       | 扩展建议                              |
|------------|------------------------------------------------|------------------------------------------------|
| 数据管理   | 支持 CSV/Excel 等文件上传、预览、导出         | 数据库集成（SQLite/MySQL）                     |
| 数据清洗   | 缺失值处理、异常值检测                         | 自动化清洗规则配置                             |
| 可视化     | 动态生成 3 种以上图表                          | 用户自定义图表参数                             |
| 分析功能   | 至少实现 1 种分析（聚类 / 预测 / 分类 / 降维） | 多算法对比 + 效果评估                          |
| Web 界面   | 响应式布局，基础交互                           | 多用户登录、历史记录存储、检索等管理功能       |
